# Vue + Tailwind Interactive Component Examples

A small collection of interactive Vue + Tailwind examples, written from scratch.

[See the examples live on Netlify &rarr;](https://vue-tailwind-examples.netlify.com/)

## Project setup
```
yarn install
yarn run serve
```

Or

```
npm install
npm run serve
```

npm run build
